ACM Social Media Mob Mentality

The influence of social media has facilitated major trends and shaped popular culture into what it is today, affecting mob mentality among users. This project will focus on the impact of social media on mob mentality and how that affects corresponding stock trends. We hypothesize that there will be a direct relationship between the sentiment of the Netflix tweets and the NFLX stock data; as sentiment becomes more positive, stock prices will increase.

STOCK QUERYING/ANALYSIS

The NFLX stock data was retrieved from historical daily stock prices available on NASDAQ using the yFinance Python package, and we specifically looked at the closing prices between September 1, 2018 to September 1, 2021.

TWEETS QUERYING

  • 90 tweets a day from 9/1/2018 to 9/1/2021 using the #Neflix tag on Twitterta
  • Implemented a custom tweet scraper by using Puppeteer
    • Puppeteer is a Node Library that allows automation of the chrome browser to run website tests.
    • The scraper opens up the URL for the #netflix tweet on a date, grabs datesand text body.

SENTIMENT ANALYSIS

Upon extracting the tweets, we needed to generate sentimental scores which will be plotted on a visualizer to trace for correlation between sentiment scores and stock market prices. In order to get accurate sentiment scores we first need to clean the data to remove data that will reduce the effectiveness of the sentiment analyzer.

Data Cleaning
  • We retained: Original casing, content of the hashtag, punctuation
  • We removed: inks, URLs, digits, mentions of other users using python regular expressions Stopwords using the python Gensim library
Sentiment Analysis

We used VADER (Valence Aware Dictionary and Sentiment Reasoner) as our primary tool to conduct sentiment analysis.

  • Assigns a tweet a value between -1 and +1.
  • Scores of +1 being extremely positive and -1 extremely negative.

GRAPHING DATA

We proceeded to make an accurate visualization of the stock market closing prices and sentiment scores after extracting the two types of data.

We graphed two different graphs
  1. The stock market closing prices for the past three years
  2. The average sentiment score of all tweets each day for the past three years

We used the Bokeh python library for this process

Results

Sentiment score distribution

  1. 97% of the tweets were categorized as positive sentiments
  2. 2.7% of the tweets were categorized as neutral sentiments
  3. 0.3% of the tweets were categorized as negative sentiments

Procedure of calculating correlation

In order to interpret the data, we needed to calculate the correlation between the stock prices and the sentiment value for each year.

  1. Assigned a value of 1 or 0 to a pair of data values for each day: the change in stock price and sentiment relative to the sentiment mean.
    • 1 was assigned to either:
      1. a positive change in stock prices and a sentiment greater than the sentiment mean
      2. a negative change in stock prices and a sentiment less than the sentiment mean
    • 0 was assigned to the other comparisons.
  2. The sum of these values were used to compute an average correlation scorefor each year with was 0.457

Conclusion

The interpretation of the correlations score is as follows:

  • Due to a designation of four possible outcomes when rating each pair of data values and two outcomes corresponding to each 1 and 0, if assigned at random, it follows:
    • 1 would correspond to roughly half of the total pairs and 0 for the remainingpairs.
    • The expected value of this random distribution is 0.5
  • An average correlation score closer to 1 suggests a strong direct while a correlation of 0 suggests inverse relationship

Therefore, the overall score of 0.46 falling within a close range of 0.5 implies a random correlation between the changes in NFLX stock prices and the changes in the sentiment of the Netflix tweets
